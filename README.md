# RL Self-Driving Car (BeamNG.ai)

This project uses Reinforcement Learning algorithms (starting with PPO) to train a self-driving car in a simulated environment using BeamNG.tech. Key scenarios include:
- Lane-following
- Obstacle avoidance
- Traffic rule compliance
